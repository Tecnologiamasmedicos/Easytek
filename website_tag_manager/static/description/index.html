GOOGLE TAG MANAGER

Características del módulo

El modulo permite agregar el tag manager de google a los diferentes sitios web que se tenga


Configuracion

Para agregar el GTM-ID se debe ir a sitio web/configuración/ajustes y elegir el sitio web al cual desea configurar


agregar GTM-ID

Nos dirigimos a la parte donde dice Google Tag Manager, marcamos el ckeck y luego agregamos el codigo el GTM-ID
