Envío de código para verificar el correo del cliente

Decripción del módulo

Este modulo lo que hace es enviar un código al cliente para verificar el correo a la hora de hacer una compra de un producto.


Configuración

Elegir un servidor de correo saliente del cual Odoo va a enviar el código de verificación al cliente.
